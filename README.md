## 🏆 Dự Án Nổi Bật

### 1. **☕ Hệ Thống Quản Lý Quán Cà Phê - Microservices Architecture**
**Repository**: [coffe-shop](https://github.com/hoanggiakz/coffe-shop)

**Mô tả**: Hệ thống POS (Point of Sales) toàn diện cho quán cà phê với 3 nhóm người dùng (Khách, Nhân viên, Quản lý).

**Kiến trúc & Công nghệ**:
- 🏗️ **8 Microservices** (NestJS + Spring Boot)
- 🔐 API Gateway với RBAC theo role
- 💬 Chat realtime qua Socket.IO
- 📊 Dashboard analytics & reports
- 💳 Thanh toán online (SePay) & cash
- 🎯 QR Table, KDS (Kitchen Display System)
- 🛢️ Redis caching, PostgreSQL, Kafka event streaming
- 📦 Orchestration với Kubernetes manifests

**Các Service**:
- `api-gateway` - NestJS proxy/auth layer
- `user-service` - Spring Boot: Auth, Staff, Branch management
- `order-service` - NestJS: Menu, Orders, KDS, Promotions
- `chat-service` - NestJS: Realtime chat + notifications
- `payment-service` - NestJS: Thanh toán (CASH/SePay)
- `inventory-service` - NestJS: Kho hàng, stock management
- `report-service` - NestJS: Analytics & business reports
- `table-service` - Spring Boot: Bàn, QR, gọi phục vụ

**Điểm Nổi Bật**:
- ✅ Multi-tenant: Hỗ trợ nhiều chi nhánh
- ✅ Realtime: WebSocket chat, KDS updates
- ✅ Event-driven: Kafka event streaming
- ✅ Security: JWT, RBAC, input validation
- ✅ Scalability: Docker Compose + K8s ready
- ✅ Testing: Unit tests, integration tests
- ✅ Documentation: API docs, deployment guides
