# 👋 Xin chào, mình là Hoàng Gia

Mình là sinh viên yêu thích lập trình và đã thực hiện nhiều bài tập/dự án nhỏ về hướng đối tượng và ứng dụng thực tế.

## 🧠 Kỹ năng & kiến thức đã sử dụng
- Lập trình hướng đối tượng (OOP): kế thừa, đa hình, lớp trừu tượng
- Thiết kế class/model dữ liệu cho bài toán quản lý
- Làm việc với cấu trúc dữ liệu Java: `ArrayList`, `LinkedList`
- Xây dựng ứng dụng Java console có menu thao tác
- Xây dựng giao diện web tĩnh bằng HTML/CSS/JavaScript

## 💻 Ngôn ngữ & công nghệ tiêu biểu

- HTML5
- CSS3
- JavaScript
